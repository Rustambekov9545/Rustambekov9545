## 🧠 Мой “интерфейсный контракт”
- **Кнопки не прыгают**: стабильный layout, предсказуемые состояния
- **Клавиатура не плачет**: focus, aria, tab-order
- **Скорость не на словах**: меньше JS, умный split, кеш, prefetch
- **Код читается**: типы, линтеры, маленькие компоненты, понятные PR

---

## 🧩 Что делаю в проектах
- UI kit и дизайн-системы, токены, темы, компонентные контракты
- Next.js приложения: SSR/SSG/ISR, роутинг, авторизация, middleware
- Данные и состояние: React Query, Zustand или Redux по ситуации
- Тесты: Playwright e2e, RTL, smoke и регресс на критичных флоу
- Полировка: анимации, skeleton, empty states, ошибки без паники
